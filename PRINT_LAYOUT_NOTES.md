# Print Layout Notes

## Landscape Settings

- ใช้ `@page { size: A4 landscape; margin: 10mm; }`
- ใช้ jsPDF แบบแนวนอน `l`
- ตารางรายเดือนใช้ขนาดฟอนต์เล็กลงและกำหนดความกว้างคอลัมน์รายวันแบบคงที่

## การจัดหัวรายงาน

- เปลี่ยนจาก header แบบ dashboard ไปเป็น header แบบเอกสาร
- แสดง:
  - ชื่อโรงพยาบาล
  - ชื่อระบบ
  - ชื่อรายงาน
  - subtitle ตาม filter ปัจจุบัน
  - ผู้จัดทำ
  - เวลาพิมพ์
  - รูปแบบเอกสารแนวนอน

## Metadata ที่ย้ายขึ้น header

- เอา `เวลาที่สร้างรายงาน` ออกจาก body card เดิม
- เอา `จัดทำโดย` ออกจาก body card เดิม
- รวม metadata ไว้ใน header block เดียวกันทั้งหมด

## Footer / Signature

- เพิ่ม footer block สำหรับ:
  - legend รหัสเวร
  - ข้อความหมายเหตุ
  - พื้นที่ลงชื่อผู้ตรวจสอบเวร

## ความต่างระหว่าง Daily กับ Monthly

- Daily:
  - ใช้ตาราง roster ปกติ
  - ยังแสดงกลุ่มเวรเป็นข้อความเต็ม
- Monthly:
  - ใช้ matrix table แบบกว้าง
  - 1 แถวต่อเจ้าหน้าที่
  - 1 คอลัมน์ต่อวัน
  - ใช้รหัส `ช / บ / ด / BD`
