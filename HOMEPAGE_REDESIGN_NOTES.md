## Visual direction

- Palette chosen:
  - off-white and light gray-blue backgrounds
  - deep navy text for trust and contrast
  - teal and mint accents for a calm healthcare tone
  - soft white translucent cards for premium depth
- Styling approach:
  - restrained soft-UI influence
  - rounded cards and buttons
  - gentle shadows
  - minimal border noise

## Hero layout decisions

- Used a two-column hero so the page feels lighter and more structured.
- Left side focuses on:
  - one strong headline
  - one short support line
  - two clear CTA buttons
- Right side uses a soft operations panel instead of decorative wellness visuals.
- The support panel highlights the three core system jobs:
  - ลงเวลาเวร
  - ตรวจสอบ
  - รายงาน

## CTA hierarchy

- Primary CTA:
  - `เข้าสู่ระบบ`
- Secondary CTA:
  - `สมัครใช้งาน`
  - `ดูคุณสมบัติระบบ`
- Button styling decisions:
  - primary button uses a deeper blue gradient and stronger shadow
  - secondary buttons use soft white surfaces and lighter emphasis
  - final CTA section uses inverted button treatment for contrast

## Feature card structure

- The homepage uses 4 concise feature cards only:
  - ลงเวลาเวร
  - ตรวจสอบรายการ
  - รายงานและส่งออก
  - สิทธิ์ตามบทบาท
- Each card contains:
  - one icon
  - one short title
  - one short support line
- Copy was intentionally reduced to keep the page scan-friendly.

## How the reference inspiration was adapted

- Borrowed the calm, premium, soft-surface feeling from wellness-style landing pages.
- Removed consumer-style content patterns such as download prompts, testimonials, and lifestyle messaging.
- Translated the mood into a hospital-system UI by emphasizing:
  - clarity
  - trusted tone
  - operational usefulness
  - real routes and real next actions only

## Typography and spacing decisions

- `Prompt` is used for headings to create stronger hierarchy.
- `Sarabun` remains the body font for Thai readability.
- Section copy is kept intentionally short.
- Padding and whitespace were expanded to make the homepage feel more breathable and less crowded.
